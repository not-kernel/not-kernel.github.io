KernelSU Next

A kernel-based root solution for Android

Supports kernel 4.4 - 6.6 (Non-GKI & GKI)

Features

Non-GKI kernel support

Non-GKI kernel support from 4.x - 5.4 LTS (3.x is experimental).

GKI kernel support

GKI kernels support LKM/KMI methods for easy installation.

Dynamic module mount

Includes both Magic Mount and OverlayFS, which can be switched from the settings with a single toggle.

New manager features

Manager app has a new, redesigned look with several feature improvements and a UI overhaul.

Module backup & restoration

Allows for backup and restoration of modules, whether they were uninstalled accidentally, you changed your mind, or you simply want to restore a specific backup.

Auto-updates

Manager app updates automatically with new releases.

Frequent updates

Frequently updated with lots of upcoming features.

Hide hosts

Hides hosts file modifications from detectors using app profile unmount, allowing you to use ad blockers without any hassle.

Bulk installation

Bulk install multiple modules at once, select all the modules you want and everything will be installed in serial.
