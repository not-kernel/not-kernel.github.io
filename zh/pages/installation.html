轻松集成

复制、粘贴并运行！轻松集成的 CLI 命令。

安装

在内核源代码目录的根目录下运行以下命令以获取不同的安装选项。

最新发行版本（稳定的）

curl -LSs "https://raw.githubusercontent.com/rifsxd/KernelSU-Next/next/kernel/setup.sh" | bash -

Next 分支（开发版本）

curl -LSs "https://raw.githubusercontent.com/rifsxd/KernelSU-Next/next/kernel/setup.sh" | bash -s next

指定版本标签

curl -LSs "https://raw.githubusercontent.com/rifsxd/KernelSU-Next/next/kernel/setup.sh" | bash -s v1.0.3

Next-SuSFS 分支（稳定的）

curl -LSs "https://raw.githubusercontent.com/rifsxd/KernelSU-Next/next-susfs/kernel/setup.sh" | bash -s next-susfs

Next-SuSFS 分支（开发版本）

curl -LSs "https://raw.githubusercontent.com/rifsxd/KernelSU-Next/next/kernel/setup.sh" | bash -s next-susfs-dev
